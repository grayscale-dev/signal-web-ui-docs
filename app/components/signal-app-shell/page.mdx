# signal-app-shell

<LiveExample tag="signal-app-shell" />

Layout shell providing global padding and framing for apps.

## Usage
```html
<signal-app-shell padded>
  <main>Content</main>
</signal-app-shell>
```

## Props
- `padded` boolean: adds internal spacing.

## Events
None.

## Slots
- default: app content.
