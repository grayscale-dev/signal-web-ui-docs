# signal-tabs

<LiveExample tag="signal-tabs" />

Tabs/steps component with validation states and lazy rendering.

## Usage

```html
<signal-tabs id="tabs" value="one"></signal-tabs>
<script type="module">
  const tabs = document.querySelector('#tabs');
  tabs.tabs = [
    { key: 'one', label: 'One' },
    { key: 'two', label: 'Two' }
  ];
</script>
```

## Props

- `tabs` array of tab configs
- `value` / `activeKey` string
- `variant` string (e.g., "underline", "pill")
- `orientation` "horizontal" | "vertical"
- `keepAlive` boolean
- `lazy` boolean
- `validationMap` record of validation states

```ts
type TabConfig = {
  key: string;
  label: string;
  disabled?: boolean;
  badge?: string;
  status?: string;
};

type ValidationMap = Record<string, string | boolean>;
```
