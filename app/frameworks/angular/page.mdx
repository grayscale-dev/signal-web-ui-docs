# Angular wrapper

The Angular wrapper offers directives and components that align with Angular templates.

```ts
import { SignalWebUiModule } from '@signal-web-ui/angular';

@NgModule({
  imports: [SignalWebUiModule]
})
export class AppModule {}
```

```html
<signal-data-table></signal-data-table>
```

## Notes

- Add the module to your feature module or app module.
- Run the loader in your main entry before bootstrap.
