# Signal Web UI Docs

Documentation site for the Signal Web UI component system. Built with Next.js App Router, React, TailwindCSS, and MDX.

## Requirements

- Node.js 18+
- npm 9+

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

## Scripts

- `npm run dev` - start the local dev server
- `npm run build` - build the production site
- `npm run start` - serve the production build
- `npm run lint` - run Next.js linting

## Project structure

- `app/` - Next.js App Router pages
- `docs/` - future MDX content for generated docs
- `ui/` - shared UI components
- `styles/` - global styles and Tailwind layers

## Deployment

The site is static-friendly and can be deployed directly to Vercel.
